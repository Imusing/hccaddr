# HCC Address Generator

This tool allows you to encode and decode HCC addresses into images. This basically is like a QR code.


## Usage
To encode an address into an image, run:

```bash
python addrgen.py encode --message "Your HCC address here" --output "output.png"
```
To decode an address from an image, run:

```bash
python addrgen.py decode --image "output.png"
```
